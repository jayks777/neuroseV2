# NeuroseV2

NeuroseV2 é um projeto experimental focado em exploração e pesquisa em modelos neurais e pipelines de processamento de dados para tarefas de NLP e geração.

## Principais recursos
- Arquitetura modular para experimentos com modelos e datasets
- Scripts de pré-processamento e avaliação
- Configuração simples para treinos rápidos e testes

## Começando
1. Clone o repositório:

	git clone <URL-do-repositório>

2. Crie e ative um ambiente virtual (recomendado):

	python -m venv .venv
	source .venv/bin/activate   # Linux/macOS
	.venv\Scripts\activate     # Windows

3. Instale dependências:

	pip install -r requirements.txt

## Uso básico
- Ajuste os parâmetros em configs/ e execute os scripts em src/.
- Exemplos:

	python src/train.py --config configs/default.yaml
	python src/evaluate.py --checkpoint path/to/model
